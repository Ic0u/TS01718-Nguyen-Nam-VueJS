# Compiling

## americaaaa

1. **Install Live Server** (if not already installed)
   - In VS Code or similiar folks, go to Extensions (Command+Shift+X) control on windows
   - Search for "Live Server"
   - Click Install

2. **Start Live Server**
   - Right-click on `lab1.html` in your file explorer
   - Select **"Open with Live Server"**
   
   **OR**
   
   - Open `lab1.html` in the editor
   - Click **"Go Live"** button in the bottom-right corner of IDE
